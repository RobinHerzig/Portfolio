## How It's Made:

The creation of this website was fairly straightforward. Most of the work was in HTML and CSS, with a little bit of SASS and JavaScript included as well.

## Optimizations:

While this project has taught me a lot, I feel that it's mostly sterile. I would love to redo it from scratch to showcase more of my personality! I think being able to create fun and charming websites is underrated. Most of the focus is on looking as professional as possible, so when portfolios deviate from that they can really stand out!

## Lessons Learned:

Having a clear direction is important! As I build dive deeper into full-stack web development, I'm updating this portfolio to reflect my new projects. These small, incremental, updates over time have made it difficult to keep a single vision. I'm not a designer, so that's something that I find difficult.
